<h2 align="center">Self-hosting & Installation</h2>
Refer to package.json for other instructions and commands.

Prerequisites: `NodeJS >= 16`, `NPM`, `Yarn`

```bash
git clone https://github.com/CXMPUTE/dotcom && cd dotcom
yarn # Installs Yarn dependencies
# For running in a development environment:
yarn dev
# For running in a production environment:
yarn build
```
